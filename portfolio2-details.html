Django E-commerce Website Project

This project is an e-commerce web application built using Django, a high-level Python web framework. It provides essential functionalities such as user authentication, product management, shopping cart management, order processing, and more.

  • User Management: Users can securely register and log in. Email verification is required to activate accounts, ensuring legitimacy and security.
  • Enhanced Account Security: The admin panel is protected by changing the default login path. A honeypot mechanism logs fake login attempts, improving monitoring and protection against unauthorized access.
  • Product Variations: Products support variations such as color and size, with these fields being mandatory, offering customers a diverse range of choices tailored to their needs.
  • Client Reviews and Ratings: Users can post reviews and rate products. The system calculates average ratings and counts the number of reviews for each product, providing valuable feedback and insights.
  • Shopping Cart Functionality: The platform supports a dynamic shopping cart where both logged-in and guest users can add, delete, and adjust product quantities. This flexibility enhances the user shopping experience.
  • Secure Checkout Process: Checkout is restricted to logged-in users, ensuring that transactions are secure and user information is protected. This includes comprehensive management of customer data and order details.
  • Payment Integration: Integrated with PayPal, the system facilitates seamless and secure payment processing, accommodating various payment preferences and enhancing the checkout experience.
  • Static and Media Files Management: Static files (e.g., CSS, JavaScript) and media files (e.g., product images) are efficiently managed and securely served. Sensitive information within these files is protected, with planned deployment to AWS S3 for scalable storage.
  • PostgreSQL Database Integration: A robust PostgreSQL database system is used to securely store user information, product data, and order details, supporting complex queries and ensuring data integrity.
  • Future Deployment Plans: Deployment to AWS S3 for media files is on the horizon, offering scalable and reliable storage solutions, ensuring the platform can grow and meet increasing demands.

While payment integration and deployment to AWS are not yet included in this iteration of the project, I have laid a solid foundation for future enhancements and improvements. Overall, I have successfully developed a feature-rich e-commerce web application that prioritizes user experience, security, and functionality.

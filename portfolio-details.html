Django Project: Handling Import, Export, Image Compression, Bulk Email, and Email Tracking

This Django project is a comprehensive web application designed to handle various functionalities, including:

  • Import and Export Data
  • Image Compression
  • Bulk Emailing
  • Email Tracking using Celery, Redis, and Ngrok
  • Web Scraping

The project covers Django template rendering, configuration of media files, authentication, and implements security measures for sensitive information. It utilizes Celery and Redis for asynchronous task processing and Ngrok for secure tunneling. The web scraping functionality demonstrates scraping Yahoo Finance for stock analysis, providing information such as current prices and more.

  • Celery Integration: Integrated Celery for background task processing. Specifically, I used Celery to import a CSV file containing over 10 million employee details records for demonstration purposes. This allowed for efficient handling of the import process in the background, ensuring smooth operation of the application without delays or interruptions.

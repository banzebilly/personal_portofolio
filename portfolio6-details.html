Python ATM System Project

This project is a comprehensive Python-based ATM system that provides essential functionalities for managing user transactions securely and efficiently. It includes features such as user authentication, balance checking, deposits, withdrawals, money transfers, receipt generation, e-wallet creation, and transaction history.

  • User Authentication: Implemented a secure authentication system using hashed PINs with salt for verifying user identity, ensuring data protection and security.
  • Check Balance: Allows users to check their current account balance, providing real-time financial information.
  • Deposit: Enables users to deposit money into their accounts, updating the balance accordingly.
  • Withdraw: Allows users to withdraw money from their accounts, ensuring sufficient funds are available for the transaction.
  • Send Money: Facilitates money transfers between accounts, enabling seamless and secure transactions.
  • Receive Money: Allows users to receive money from other accounts, updating the balance in real-time.
  • Generate Receipt: Provides users with a receipt for their latest transaction, ensuring transparency and record-keeping.
  • Create E-Wallet: Allows users to create an e-wallet for digital transactions, offering additional convenience and flexibility.
  • Transaction History: Displays a comprehensive history of all transactions, helping users to track and manage their financial activities.
  • Secure PIN Storage: Utilizes the cryptography library for secure storage and verification of user PINs, ensuring robust data protection.

This project demonstrates proficiency in Python programming, secure data handling, and user-centric design. It offers a reliable and user-friendly interface for managing ATM transactions, prioritizing both security and usability.

Python: Financial Calculators for Investments and Home Loans

This project is a Python-based application designed for a small financial company, providing users with access to two different financial calculators: an investment calculator and a home loan repayment calculator. The application helps users calculate the amount of interest earned on investments and the monthly repayments for home loans.

  • Investment Calculator: Allows users to calculate the future value of their investments based on simple or compound interest.
    • Prompts the user to input the amount of money to be deposited, the interest rate, and the number of years for the investment.
    • Asks whether the interest is "simple" or "compound" and calculates the total amount accordingly using the formulas:
      • Simple Interest: \( A = P(1 + r \times t) \)
      • Compound Interest: \( A = P \times (1 + r)^t \)
    • Outputs the total amount after the given period at the specified interest rate.
  • Home Loan Repayment Calculator: Enables users to calculate the monthly repayment amount for a home loan.
    • Prompts the user to input the present value of the house, the annual interest rate, and the number of months for repayment.
    • Uses the bond repayment formula to calculate the monthly repayment:
      • \( repayment = \frac{i \times P}{1 - (1 + i)^{-n}} \)
    • Outputs the monthly repayment amount based on the user's inputs.
  • Interactive Menu: The program starts by presenting a menu to the user, allowing them to choose between the investment and bond calculators:
    • 1. Enter 'investment' to calculate the amount of interest earned on an investment.
    • 2. Enter 'bond' to calculate the monthly repayments on a home loan.
    • The user's input is case-insensitive and is validated to ensure correctness.
    • If an invalid option is entered, an appropriate error message is displayed.
  • Python Implementation: The application is implemented in a Python script called finance_calculators.py with the necessary calculations and user interactions.
    • Imports the math module for mathematical operations.
    • Includes clear and concise code to perform the required calculations based on user inputs.

This project effectively demonstrates the use of Python for financial calculations, providing a user-friendly interface for managing investment and loan repayment calculations. The implementation showcases practical applications of mathematical formulas, user input handling, and interactive menu design.
